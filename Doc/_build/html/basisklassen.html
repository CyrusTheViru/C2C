class basisklassen.Motor(direction_channel, pwm=None, offset=True)¶

Bases: object

Motor driver class Set direction_channel to the GPIO channel which connect to MA, Set motor_B to the GPIO channel which connect to MB, Both GPIO channel use BCM numbering; Set pwm_channel to the PWM channel which connect to PWMA, Set pwm_B to the PWM channel which connect to PWMB; PWM channel using PCA9685, Set pwm_address to your address, if is not 0x40 Set debug to True to print out debug informations.
